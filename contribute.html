Contribute

We're searching for project maintainers!
Help us manage the site review process and moderate our community. For more information join our Discord server or email us.

Would you like to help make the web a bit safer? We have some options on how you can contribute to the project!

Translate our software

We have a goal of making our software (browser extensions) as accessible as possible. If you want to help us reach that goal, please translate our extensions in your language. If your language is not listed, contact us and we'll add it for you.

 Translate on Crowdin

Contribute code

If you know how to code, please have a look at our GitHub repositories. You can submit a pull request to improve our code and make it more reliable/secure.

 Contribute on GitHub

Donate on OpenCollective

OpenCollective is a platform where you can donate money towards an organization. If (eligible) expenses arise, you (contributor, team member, etc.) can get them refunded. These donations help us cover our costs like domain renewals, hosting, etc.

 Donate on OpenCollective
